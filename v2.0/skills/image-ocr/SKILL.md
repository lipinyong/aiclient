# 图片 OCR 技能

## 使用场景

- 用户提供**本地图片路径**（如 `.png`、`.jpg`、`.jpeg`、`.bmp`），要求识别图中的文字
- 截图、证件、海报、表格截图等需要提取文字的场景
- 图中文字需翻译或整理成文本

## 工作流程

1. 用户给出图片的本地路径（如 `C:\...\screenshot.png`）
2. 立刻调用 **`ocr_run`** 工具，传入 `path` 为图片路径
3. 根据工具返回的 `text`（识别出的全文）和/或 `details`（各区域文字与置信度）进行回答、整理或翻译

## 行为规范

- 始终基于 **ocr_run** 返回的识别结果作答，不要编造图中内容
- 若识别结果为空或置信度很低，如实告知用户（可能是图片模糊、非文字图或语言未包含）
- 支持中英文混合；若用户未指定语言，默认使用 `ch_sim`（简体中文）+ `en`（英文）
- 用中文回答用户

## 说明

- **当用户提供图片路径并要求识别/提取图中文字时**：必须调用 **ocr_run** 执行识别，再根据返回的 `text` 和 `details` 整理、翻译或回答。
- 路径中的引号会被自动去除，Windows 路径可直接传入。
